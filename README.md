# Sales Performance Analysis (EDA Project)#
**Project Overview**
This project focuses on Exploratory Data Analysis (EDA) of a Sales Performance dataset to understand sales trends, regional performance, and overall business insights using Python.

**Dataset Description**
The dataset includes sales-related information such as:
Region
Product Category
Sales Amount
Date / Month
Profit (if available)

**Data Cleaning**
Checked missing values
Converted date columns to datetime
Removed duplicates
Fixed column formats

**Exploratory Data Analysis**
Sales by region
Monthly sales trend
Product category performance
Correlation heatmap
Top-performing regions/products

**Visualizations**
- Bar charts
- Line charts
- Scatter plot
- Heatmaps

**Tools & Technologies**
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

**Key Insights**
- Certain regions contribute maximum revenue
- Seasonal patterns in monthly sales
- Some product categories outperform others
Jupyter Notebook
